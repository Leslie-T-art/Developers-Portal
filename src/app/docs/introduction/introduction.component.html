Introduction

Prompt Angular is a fully featured moderen multi purpose template built with Angular v13 and Bootstrap 5.1.3. Prompt Angular is crafted by following the best industry standards in order to provide speed and scalability. The clean code structure allows you to build any Angular-based web application without any hustle. It's for building responsive, mobile-first landing pages, websites with Bootstrap.

This documentation is guidelines for all users including beginner, experienced web developer, etc

Folder Structure

Extract the zip file you received after purchase and you would find the exact below files and folders: 

├── prompt
│ ├── package.json                     => Package json file
| ├── angular.json                     => Project based configuration file
│ ├── README.md                        => Introductory documentation
│ ├── src/
│ │ └── app
│ │     └── auth                       => Account module contains the implementation for login, signup, forget password, confirm mail etc
│ │     └── core                       => The core module - mainly contains the common services (singleton ones), fake backend and guards
| |     └── docs                       => Documentation pages
│ │     └── pages                      => Pages module is containing implementation of all landing pages, other pages, and home page
│ │     └── shared                     => The shared module - containing reusable components including heros, navbars, footers and many more
│ │     └── app.module.ts              => Main module for application
│ │     └── app.component.ts           => Entry point component
| |     └── app.component.spec.ts      => Testing specification file
| |     └── app.component.scss         => Styling file for Entry component
│ │     └── app-routing.module.ts      => Router implementation
| |
│ │ └── assets
│ │     └── scss                       => Scss files
│ │     └── images                     => All images
│ │
│ ├── index.html                       => The main HTML page that is served when someone visits your site
| ├── locale                           => i18N - localization files for different language
│ ├── environments                     => Contains build configuration options for particular target environments
