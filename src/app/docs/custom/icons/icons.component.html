Icons

Prompt comes with Feather Icons. Feather is a collection of simply beautiful svg based open source icons. Each icon is designed with an emphasis on simplicity, consistency, and flexibility. To use an icon on your page, add a data-feather attribute with the icon name to an element.

For a complete list of icons, check here.

Use modifier class .icon-dual to convert it into two-tone. All the color variations are available as well. E.g. icon-dual-{{'{'}}primary|secondary|success|danger|info|warning{{'}'}}.

Use size modifier class .icon-{{'{'}}xxs|xs|sm|md|lg|xl|xxl{{'}'}} to change the size.
